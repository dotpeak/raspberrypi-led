# raspberrypi-led
2 Shell scripts to turn the power and load LEDs on a raspberrypi on or off.

## Make the files executable

Run the command below to make the files executable.

```
chmod +x on-led && chmod +x off-led
```

## Toggling the LED on

```
./on-led
```

## Toggling the LED off

```
./off-led
```
